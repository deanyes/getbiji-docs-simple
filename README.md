# Get笔记 使用文档

> 生长在 AI 时代下的小产品，让**记笔记、找笔记、用笔记**变得简单。

## 核心功能

| 功能 | 说明 |
|------|------|
| 🎙️ 语音速记 | 60分钟超长录音，27种方言识别 |
| 📸 拍照记录 | 拍书、白板、文档自动 OCR |
| 🔗 链接收藏 | 文章、视频一键转笔记 |
| 🔍 AI 搜索 | DeepSeek-R1 满血版 |
| 📚 知识库 | 碎片变系统，支持团队协作 |
