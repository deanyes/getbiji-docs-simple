# 导入音视频

> 导入音视频上传，让冗长内容一键整理！

把长达 3 小时的课程、会议或演讲音视频文件直接上传，让 AI 在后台完成转写与总结，从此告别繁琐手动整理。

## 适用场景

- **线上 & 线下课程**：将老师或讲师的整段授课录音转写为笔记，方便后期复习与检索
- **商务会议 / 研讨会**：上传录制文件，迅速产出结构化纪要，节省整理时间
- **专题访谈 / 播客**：还原对话原貌，按时间轴转写文字稿，便于后期创作与归档
- **语音备忘录**：平时录下的灵感或想法，自动转写为可搜索文本并智能汇总

## APP 端使用

1. 打开 App，点击首页底部+号，选择「导入音视频」

::: tip 支持格式
支持主流音频 / 视频格式（MP3、WAV、M4A、MP4、MOV、AVI 等）
:::

2. 可以从相册、本地文件中导入音视频内容

3. 上传文件后，AI 会分析文件内容选择最佳的模板进行总结整理

4. 完成后，AI 会生成音频对应的转写文字稿和总结笔记

## 从语音备忘录导入

iOS 用户可以从「语音备忘录」App 分享音频到 Get笔记：
1. 打开系统「语音备忘录」App
2. 长按或点击要导入的录音
3. 点击分享按钮
4. 选择「Get笔记」即可导入

## 转写额度 & 文件时长

转写时长额度月初重新计算不累计，下月自动更新，PRO 会员到期后自动降为普通额度。

## 网页端使用

- 首页 - 导入音视频
- 知识库 - 添加 - 导入音视频
- 追加笔记 - 导入音视频
