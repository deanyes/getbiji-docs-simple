# 链接一键记

> 粘贴链接，AI 帮你提取精华

支持多种平台链接，AI 自动读取内容并总结重点。

## 支持的链接类型

- **图文链接**：微信公众号文章、小红书图文、公开免费网站等
- **视频链接**：抖音短视频、B站视频、小红书视频、得到直播回放（需要使用得到App中的分享链接）等

## 粘贴公众号文章链接

1. 点击一篇公众号文章，在右上角...中选择复制链接
2. 打开 Get 笔记 APP，点击底部 + 号，选择"粘贴链接"
3. 粘贴链接后，可以在链接下方添加补充信息和要求，比如"整理 xxx 方面的内容"
4. 最后，点击"确定"，AI 会读取文章内容并帮你记录总结重点信息

::: tip 图片理解
若链接中包含多张图片，AI可理解到10张图，且图片理解的能力也会随AI能力的发展逐步增强
:::

## 粘贴抖音短视频链接

1. 点开一个抖音短视频，在右下角点击分享按钮，选择"分享链接"，然后复制链接
2. 打开 Get 笔记 APP，点击底部 + 号，选择"粘贴链接"
3. 粘贴链接后，AI 会自动提取其中的 url，同时可以在链接下方添加补充信息和要求，比如"整理 xxx 方面的内容"
4. 最后，点击"确定"，AI 会读取短视频内容并帮你记录总结重点信息

## 粘贴小红书图文/短视频链接

1. 点开一个小红书图文/短视频，在点击分享按钮，然后复制链接
2. 打开 Get 笔记 APP，点击底部 + 号，选择"粘贴链接"
3. 粘贴链接后，可以在链接下方添加补充信息和要求，比如"整理 xxx 方面的内容"

::: tip 图片文字识别
如果图片中有文字，也会识别图片中的文字，并进行整理，同时会把图片也保存下来
:::

4. 最后，点击"确定"，AI 会读取图文/短视频内容并帮你记录总结重点信息

## 粘贴 B 站视频链接

1. 点开一个 B 站视频，再点击分享按钮，然后复制链接
2. 打开 Get 笔记 APP，点击底部 + 号，选择"粘贴链接"
3. 粘贴链接后，可以在链接下方添加补充信息和要求，比如"整理 xxx 方面的内容"
4. 最后，点击"确定"，AI 会读取短视频内容并帮你记录总结重点信息

::: warning 温馨提醒
如果视频时长过长（如2小时），AI 处理等待时间会较长，可以先收起处理页面，继续进行其他操作，AI 依然会在后台继续处理。
:::

## 粘贴得到直播回放链接

1. 在得到APP内选择一场公开直播回放，再点击分享按钮，然后复制链接
2. 打开 Get 笔记 APP，点击底部 + 号，选择"粘贴链接"
3. 粘贴链接后，可以在链接下方添加补充信息和要求，比如"整理 xxx 方面的内容"
4. 最后，点击"确定"，AI 会读取直播回放内容，并帮你记录总结重点内容、章节纪要
