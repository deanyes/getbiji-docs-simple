# GetSeed 录音卡

> 于点滴记录里，听见生长

## 什么是 GetSeed？

GetSeed，Seed 是种子。每一次记录，都是播下一颗知识的种子。你播种，它培育，让知识在你身上生根、发芽、生长。

大多数录音产品停留在"帮你记住"——录音、转写、存档。笔记越存越多，却变成了信息的坟墓。

我们相信，记录的真正价值不是"记住"，而是"生长"。

## 核心功能

### 一键开启录音
- **打开即录**：拨动开关，1秒开始录音。再拨一下，结束录音。无需解锁、无需找APP
- **一键 Mark 重点**：短按 Mark 键，快速标记重点时刻，会后 AI 自动生成重点摘要
- **多模态记录**：录音中可拍照、随手记，会后 AI 结合所有内容生成图文并茂的笔记

### 通话录音
电话、微信视频、腾讯会议、钉钉、飞书，全都能录。
- 普通电话 ✓
- 微信语音/视频通话 ✓
- 腾讯/钉钉/飞书会议 ✓
- 现场录音 ↔ 通话录音，智能识别，自动切换

### 闪念随时记
长按 Mark 键，快速记录灵感。AI 自动转写、智能润色、去除口水词、识别方言。

支持 12 种方言：普通话、英语、粤语、四川话、陕西话、河南话、上海话、湖南话、湖北话、安徽话、闽南话、潮汕话

## 为什么需要 GetSeed？

| 痛点 | GetSeed 解决方案 |
|-----|-----------------|
| 手机录音要解锁、找APP | 盲操一秒开录，物理按键拨动即录 |
| 来电/消息打断录音 | 独立运行，手机来电消息都不影响 |
| 手机没电不敢录 | 35 小时连续录音，待机 60 天 |
| 通话录不了 | 电话、微信、会议软件全都能录 |
| 没网录不了 | 64GB 本地存储，录完再同步 |

## 适合谁？

- 📋 **会议缠身的职场人**：产品经理、销售、项目经理、管理者
- 📚 **终身学习的知识行动派**：考研/考公/考证党、MBA学员、培训常客
- ✍️ **深度的内容创作者**：自媒体博主、播客主播、作家/编剧
- 💼 **需要留存沟通记录的专业人士**：律师、咨询顾问、记者、创业者
