# 创建知识库

> 从碎片到系统，打造你的第二大脑

## APP端使用说明

1. 从 APP 首页顶部"更多"入口进入

2. 点击知识库页面右上角"+新建"按钮

::: tip 容量说明
- 每个用户可免费创建 **10个** 知识库，空间共 **30G**
- PRO会员可创建知识库数量 **不限**，空间共 **500G**
:::

3. 创建后，即可在知识库中上传文件、订阅直播、订阅博主或添加已有笔记

4. 可以编辑知识库名称、描述，如果删除知识库，则知识库及其中的文件内容也会一并删除，请谨慎操作

## 网页端使用说明

1. 打开网页端访问地址：[biji.com](https://biji.com)

2. 已开通内测的用户，可以看到左侧菜单上的"知识库"入口，点击进入知识库页面

3. 点击创建知识库

::: tip 容量说明
- 每个用户可免费创建 **10个** 知识库，空间共 **30G**
- PRO会员可创建知识库数量 **不限**，空间共 **500G**
:::

4. 如果想要编辑、删除知识库，可以点击"..."操作
