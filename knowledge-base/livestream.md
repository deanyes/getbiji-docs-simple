# 帮你听直播

> 🎧 Get 笔记帮你听直播，知识不再错过

只需将直播链接或二维码上传，AI 就能替你全程聆听并记录，生成清晰的内容总结和章节纪要。

错过直播也无需担忧，你可以随时回看直播回放或查看全文稿，快速掌握核心内容，让每一场高质量讲座都为你所用。

## APP端使用说明

1. 创建一个个人知识库，在知识库内，点击"添加内容"按钮，并选择"订阅直播/博主"

2. 可以支持订阅 **得到App直播** 和 **视频号直播**

3. **得到App直播**：可以订阅未开始、进行中和回放的得到 APP 直播，直接粘贴直播链接即可订阅

::: warning 注意
- 已下架的直播，不支持订阅
- 会员专享的直播，不支持订阅
:::

4. **如何找到得到 APP 的直播链接？**
   - 打开得到 App，找到想要订阅的直播
   - 点击直播页面右上角「分享」按钮
   - 选择「复制链接」
   - 将链接粘贴到 Get 笔记即可

5. 直播结束后，Get 笔记会帮你生成清晰的内容总结和章节纪要（这个过程 AI 需要 30-40 分钟的分析处理）

## 网页端使用说明

1. 进入你创建的知识库，点击"添加"按钮，选择"订阅直播/博主"

2. 目前支持订阅 **得到APP直播**

3. 得到APP直播，可以订阅未开始、进行中和回放的得到APP直播，直接粘贴直播链接即可订阅

::: warning 注意
- 已下架的直播，不支持订阅
- 会员专享的直播，不支持订阅
:::
