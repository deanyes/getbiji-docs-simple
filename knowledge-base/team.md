# 分享和团队管理

> 📤 分享订阅，知识触手可及

支持将知识库以多种形式（微信小程序、分享海报、链接等）分享给他人，也可订阅其他人的「靠谱知识库」，获取独家内容和实时更新。共享智慧，扩展视野，让学习更高效！

同时，新增了知识库的团队管理功能，可以让知识库在限定范围内进行共创和使用，大大提高了知识库的共创和灵活性。

## APP端使用说明

个人自己创建的知识库，默认是私密状态，仅自己可见。

如果想要分享给他人查看，或邀请团队成员共同使用，需要手动设置知识库的分享权限。

**权限包括：**

- **公开**：获得你的知识库分享链接或分享海报的人，都可以访问知识库使用（AI搜索、查看内容）
- **团队**：获得你的知识库分享链接或分享海报的人，无法直接访问知识库内容，需要先申请加入才能使用

::: tip 团队权限说明
创建人可以自定义成员加入时，是否需要管理员审核通过后才能加入
:::

## 分享自己的知识库

1. 在知识库内，点击右上角分享按钮

2. 选择"公开"，然后将知识库通过「微信好友」、「朋友圈」、「分享海报」和「分享链接」等形式分享出去

3. 他人在访问你的知识库时，可以直接进入知识库来使用

## 订阅他人公开的知识库

1. 点击他人分享给你的知识库小程序、海报或链接

2. 点击进入知识库页面，可以点击"+订阅"，订阅后的知识库，会出现在「我的」知识库列表内

3. 同时，你也可以在精选 tab 下，点开官方精选知识库，并点击"订阅"按钮，完成订阅

4. 如果想取消已订阅的知识库，可以再次点击即可"取消订阅"

## 设置团队知识库

1. 在知识库内，点击右上角分享按钮

2. 选择"团队可见"，然后将知识库通过「微信好友」、「朋友圈」、「分享海报」和「分享链接」等形式分享出去

::: warning 审核设置
默认"成员自动加入"是关闭的，即加入的成员需要管理员审核通过才能访问知识库。其他成员点击进入知识库页面，可以点击"申请加入知识库"，然后等待知识库管理员的审核即可。
:::

3. 管理员审核同意后，加入后的知识库，会出现在其他成员「我的」知识库列表内

4. 如果打开"成员自动加入"，则不需要任何审核确认，成员可以直接加入并使用知识库

### 审核加入团队知识库申请

1. 有新的成员申请加入时，知识库顶部成员入口会有红点提醒，点击"成员"入口进入成员管理页面

2. 可以对申请的成员进行审核，可以同意加入，也可以拒绝，同意加入则进入成员列表，可以分配具体角色权限

3. 可以将其设置为普通成员和管理员，具体请参考下面的【角色权限说明】表格

4. 同时，可以将已加入的成员，从知识库移除，则其无法再正常访问和使用知识库

## 网页端使用说明

### 分享自己的知识库

1. 进入知识库内，点击右上角"分享"按钮，选择"公开"后，然后复制链接给他人，即可实现分享

2. 如果不想其他人能访问到此知识库，则关闭此开关，则他人无法继续访问和使用你的知识库

3. 如果想要团队共同维护一个知识库，则选择"团队"，即可由团队管理员共同维护此知识库

### 订阅他人的知识库

1. 打开别人分享给你的知识库，点击右上角"订阅知识库"

2. 订阅成功后，该知识库会出现在你的知识库列表内

3. 如果想取消已订阅的知识库，可以点击他人知识库的 ... 更多入口，选择"取消订阅"

## 小程序端使用说明

1. 进入 Get 笔记小程序，点击右上角更多，可以进入知识库列表页

2. 选择要访问的知识库，然后在知识库内，点击右上角分享按钮

3. 选择"公开"，可以将知识库通过「微信好友」、「朋友圈」、「分享海报」和「分享链接」等形式分享出去

   则他人在访问你的知识库时，可以直接进入知识库来订阅、使用

4. 如果想要团队共同维护一个知识库，则选择"团队"，即可由团队管理员共同维护此知识库
