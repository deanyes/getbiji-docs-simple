# 上传文件

> 📚 上传文件，轻松管理知识

支持 PDF、Word、PPT 格式文件，一键上传，AI 自动分类整理，智能 AI 搜索，随时调用，让知识更系统、更高效！

## APP端使用说明

1. 在知识库内，点击"添加内容"按钮，并选择"上传文件"

::: tip 支持格式
目前支持 PDF、Word、PPT 等格式
:::

2. 单个文件，限制 **200MB** 以内

3. 普通用户可以创建10个知识库，一共可容纳 **30GB** 的内容；PRO会员知识库创建数量不限，总共可容纳 **500GB** 的内容

4. 同时可以支持"新建文件夹"，将上传的文件，放入某一文件夹中进行分类

## 网页端使用说明

1. 进入你创建的知识库，点击"添加"按钮，直接点击上传文件即可（可多选），目前支持 PDF、Word、PPT 等格式

   单个文件，限制 **200MB** 以内

2. 普通用户每个知识库，可容纳 **10GB** 的内容；PRO会员知识库创建数量不限，总共可容纳 **500GB** 的内容

3. 上传后，AI 需要对文件进行识别分析和处理，这个过程一般需要 1-3 分钟，可以继续添加其他文件，不影响正常操作

4. 点击具体文件，可以查看文件内容

## 小程序端使用说明

1. 在微信下，打开 Get 笔记小程序，点击"更多"按钮，进入知识库页面

2. 点击右下角添加按钮，可以选择"微信文件"，则可以上传导入微信聊天中的文件

::: tip 支持格式
目前支持 PDF、Word、PPT 等格式，单个文件限制 **200MB** 以内
:::

3. 也可以支持微信下打开一个文件，然后点击右上角...入口，然后点"在小程序工具中打开"，选择"Get笔记"，就会自动打开 Get 笔记小程序

4. 在添加弹窗中，选择具体某个知识库，就可以把文件加入到此知识库下
