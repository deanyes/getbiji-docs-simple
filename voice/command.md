# 语音口令

> 语音口令——录音中说句话，AI 帮你标记重点

录音时说出口令词，AI 自动捕捉关键内容并特别处理

Get笔记全新推出语音口令功能，你可以自定义口令词和对应的处理要求。录音过程中，只需说出口令词，AI 就能自动识别并对相关内容做特别处理——标记重点、提炼金句、总结要点，动动嘴就能完成。

## 适用场景

- **课堂听讲**：老师讲到重点时，说一句「帮我记一下」，AI 自动捕捉该知识点并总结，不用暂停、不用切 App、不用打字，2小时的课10分钟就能复习完
- **读书思考**：看到精彩段落念出来，说一句「这段不错」，手不离书、眼不离页面，看完后读书笔记自动整理好
- **灵感捕捉**：开车、通勤时突然冒出好想法，说一句「这个想法记一下」，到了目的地灵感全都在，不会再丢了
- **听播客/音频节目**：一边听一边录，听到有启发的观点说一句「这个有启发」，一期听完金句自动整理好——把「消费型听」变成「积累型听」
- **语音日记/个人复盘**：每天录一段语音回顾，哪些特别想记住的说一句「这个很重要」，AI 生成日记时会把你标记的内容作为重点——重点由你自己决定
- **带娃记录**：孩子突然说了句可爱的话，你正在忙没法掏手机，说一句「这句真可爱」就能记下来，等孩子长大这些就是最珍贵的回忆

## 核心功能

- **自定义口令词**：支持设置最多 3 个口令词，每个口令词可指定不同的处理要求（如「帮我记一下」→ 捕捉知识点并总结，「这段不错」→ 提炼金句或精彩表达）
- **实时识别**：录音过程中 AI 实时监听口令词，无需手动操作
- **智能处理**：识别到口令词后，AI 自动对对应的录音内容按你设定的要求进行处理
- **标记整合**：口令词触发的处理结果统一展示在笔记的「标记」tab 中，同时在智能总结中重点标注，方便快速回顾

## APP端使用说明

1. 打开 App，进入「我的账号」页面，在常用操作区域点击「语音口令」

2. 进入语音口令设置页，开启「开启语音口令功能」开关

3. 点击「添加口令词」，输入你想使用的口令词（如「帮我记一下」），并填写对应的处理要求（如「捕捉当下讲到的知识点并总结」）

4. 最多可添加 3 个口令词，设置完成后即可生效

5. 正常使用录音功能，录音过程中需要标记时，直接说出口令词即可——AI 会在生成笔记时自动识别并处理

6. 录音结束后，进入笔记详情，在「标记」tab 中查看所有口令词触发的处理结果，智能总结中也会对这些内容做重点标注
